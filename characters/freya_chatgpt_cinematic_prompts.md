# Freya — ChatGPT Creative / Cinematic Prompts

Reflection, shadow, silhouette, and urban shots. These make the feed feel curated and real rather than generated. Mix these in between standard content posts.

**Usage notes:**
- Prompts marked *[no face]* can be generated in any fresh ChatGPT session — no consistency needed
- All others should be run in the same session as prompt #1 in `freya_chatgpt_prompts.md`
